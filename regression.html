3 Testing the equality of multiple coefficients

3.1 Method 1: (Paternoster et al (1998))

In this example we want to test the hypothesis \(H_0: \beta_1 = \beta_2\) against otherwise. Thus, after extracting the intercept coefficient, we built the statistics as discussed in the handbook.

bs <- coef(model1)[-1]
V <- vcov(model1)[-1, -1]
tibble::tibble(
  diff_estim = diff(bs),
  diff_SE = sqrt(V[1, 1] + V[2, 2] - 2 * V[1, 2]),
  t_stat = diff_estim / diff_SE,
  df = df.residual(model1),
  p_value = 2 * pt(abs(t_stat), df = df, lower.tail = FALSE)
)
## # A tibble: 1  5
##   diff_estim diff_SE t_stat    df p_value
##        <dbl>   <dbl>  <dbl> <int>   <dbl>
## 1       2.03   0.899   2.26    12  0.0436

\(P-value\) is barely less than the assumed \(\alpha\) (\(0.05\)), consequently not accepting the hypothesis of equality of the coefficients. However, this hypothesis could be accepted in \(\alpha = 0.01\).

3.2 Method 2: By R package

The desired hypothesis could be more easily tested using \(car\) package. The command \(linearHypothesis\) does so. One just needs to specify the model and pass in the desired hypothesis in \(c("\space" )\).

library(car)
linearHypothesis(model1, c("x1 = x2" ))

6 Influential Observations

6.1 Cooks distance

Cooks distance or Cooks D is a commonly used estimate of the influence of a data point when performing a least-squares regression analysis. A general rule of thumb is that any point with a Cooks Distance over \(\frac{4}{n}\) or \(\frac{4}{n-k-2}\) (where \(n\) is the total number of data points and \(k\) is number of parameters) is considered to be an outlier.

One can investigate the values of cooks distance using the following code.

7 High-leverage points

Observations will be considered as high-leverage points if they resemble outliers when compared to other predictors. Strictly speaking, they have uncommon combination of predictor values while the response value has minor impact on determining leverage. You can compute the high leverage observation by looking at the ratio of number of parameters estimated in model to the sample size. If an observation has a ratio greater than \(2-3\) times the average ratio, then the observation is considered as high-leverage points. We use the below function to identify high-leverage observations.

highleverage <- function(fit) {
  p <- length(coefficients(fit))
  n <- length(fitted(fit))
  ratio <-p/n
  Index = as.numeric(which(hatvalues(fit)>2*ratio))
  hatvaluesleverage = hatvalues(fit)[Index]
  plot(hatvalues(fit), main='Index Plot of Ratio')
  abline(h=c(2,3)*ratio, col='red', lty=2)
  text(hatvaluesleverage ~ Index , labels=names(hatvaluesleverage),data=states, cex=0.9, font=2)
}
highleverage(fit)

The graph above indicates that Alaska and California have the highest ratio. It means these two states are quite unusual if we compare their predictor values to other states. Here is why:

  • For Alaska, this state has a much higher income compared to other states, while having smaller population and lower temperature.
  • For California, this state has a much bigger population than other states, while having higher income and higher temperature. Both of these states are indeed quite particular compared to the other 48 states. One thing to remember is that not all high-leverage observations are influential observations. It will depend on if they are outliers or not.

12 The Durbin Watson (DW) statistic

Durbin watson statistic is a test for autocorrelation in the residuals from a statistical model or regression analysis. The Durbin-Watson statistic will always have a value ranging between 0 and 4. A value of 2.0 indicates there is no autocorrelation detected in the sample. Here the \(H_0: There\space is\space no\space correlation\space among\space the\space residuals\) and \(H_1: The\space residuals\space are\space autocorrelated\).

durbinWatsonTest(lm(fconvict ~ tfr + partic + degrees + mconvict, data=Hartnagel))
##  lag Autocorrelation D-W Statistic p-value
